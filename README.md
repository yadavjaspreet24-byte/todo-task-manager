ToDo Task Manager

This project is a simple task management website that helps users keep track of the work they need to do. It allows users to add new tasks, mark tasks as completed, and delete tasks when they are finished. The goal of the project is to provide a clean and easy way for anyone to manage their daily activities.

Features
Add new tasks using a simple input box
Mark tasks as completed to keep track of progress
Remove tasks that are finished
Clean layout that is easy to understand
Works directly in any web browser
How the Project Works

The website is made using three main files:

HTML file – This builds the structure of the page.
CSS file – This controls the style, colors, and layout.
JavaScript file – This handles all the actions like adding and deleting tasks.

When a user adds a task, it appears on the page. When they click to complete or delete it, the JavaScript updates the list so the user always sees the latest changes.

Purpose of the Project

The purpose of this project is to practice and demonstrate basic web development skills using HTML, CSS, and JavaScript. It also shows how simple interactive features can be created without using any external frameworks.

How to Use
Open the main HTML file in a browser.
Type a task in the input area.
Click the add button to display the task.
Use complete or delete options to manage your list.
Technologies Used
HTML for structure
CSS for styling
JavaScript for functionality
